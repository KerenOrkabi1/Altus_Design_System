# Host Shell Variants

## Purpose
Defines host-specific shell variants such as standard app/web and Teams.

## Standard app or web
- Uses the default base shell and optional product shell extension.
- Overview family may show a visible AI panel in the canvas region.

## Teams variant
- Ask Altus may become part of the page shell and remain persistently visible.
- The canvas family can remain the same while the surrounding shell adapts to Teams constraints.

## Rule
Host changes should not force a redesign of the canvas family unless the host changes the page purpose.
